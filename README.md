## Expenses Record Application
This is a simple Java-based desktop application that allows users to record their expenses and set a monthly spending limit. The application helps users manage their finances by tracking spending habits and providing useful insights. The overall purpose of this project is to *learn*. I chose to make a expense record app to enable manage my finances spending habits. As a bonus to completing this project, I have an application that is beneficial especially for me.

Features:
- **Expense Recording** – Users can input their daily expenses with details such as date, amount, and description.
- **Edit & Delete Expenses** – Users can modify or remove previously recorded expenses in case of errors or changes.
- **Monthly Expense Limit** – Users can set a maximum spending limit for each month.
- **Expense Summary** – View total expenses for the month and compare with the set limit.
- **Search & Filter** – Users can search expenses by date.

Goals of the project:
- Help users track and manage their monthly expenses efficiently.
- Encourage better financial habits by setting spending limits.
- Provide users with insights into their spending patterns through summaries.

Targeted Users:
- International students studying abroad who want to track and control their expenses.
- Individuals who want to monitor their daily and monthly expenses.

User Stories :
- As a user, I want to be able to record an expense, so that I can track my spending.
- As a user, I want to set a monthly expense limit, so that I can manage my budget.
- As a user, I want to view a list of my recorded expenses, so that I can review my spending history.
- As a user, I want to filter expenses by date and category, so that I can easily find specific records.
- As a user, I want to edit or delete an expense, so that I can correct mistakes or remove unnecessary records.
- As a user, I want to see a summary of my total expenses for the current month, so that I know how much I have spent.
- As a user, I want to be able to save my expense record to file (if I so choose)
- As a user, I want to be able to be able to load my expense record from file (if I so choose)
- As a user, I want to be able to add multiple Expenses to Account
